Gyazo CLI
================================================================================

Gyazo command-line uploader


Usage
--------------------------------------------------------------------------------

```sh
$ gyazo ./image.png
http://gyazo.com/f1380d79593d2aaa0fcd412511f3d3e5
```


Installation
--------------------------------------------------------------------------------

```sh
$ go get github.com/Tomohiro/gyazo
```
